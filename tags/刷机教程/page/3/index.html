中兴MDM8200类机器刷机教程(mf667-mf669)

本教程适用于 MF60/MF61/MF62, MF668A, MF667, MF669, MF70, MF668, K4505-Z, MF662等机器。

准备文件：

1．    高通QPST刷机工具

2．    固件包

3．    固件刷机引导NPRG8200.hex

4．    打开端口的工具（SCSI.exe或dc-unlocker）

5．    终端AT命令工具

第一步：打开端口

双击这个文件即可（无其他显示）或者用dc-unlocker的放大镜，

然后会出现这两个端口。

第二步：设置QPST工具

设置刷机平台

选择刷机端口

选择引导文件NPRG8200.hex 。（ 此文件必须和刷机包放在同一个文件夹 ）

第三步：刷底层文件

切换到刷底层的选卡（如图），点击“Multi-image”选项卡，选择固件的目录

点击“Advance”选项，勾选图中的选项。

点击start开始，等待下面的刷机进度完成。

第四步：刷软件

切换到 CEFS 选项，选择 分区文件 和 系统固件 ，然后选择 刷机功能 的restore applications CEFS，点击start开始。

等待刷机完成。

第五步：恢复设备

刷完的机器，还需要恢复设置，不然无法正常使用。

打开终端，（可以自己换软件，我用惯了这个）

输入

AT+ZCDRUN=F 

AT+ZCDRUN=9

然后插拔一下设备，重启就可以了
